## 🔄 איך לעדכן את האפליקציה

### דרך 1: מהדפדפן (הכי פשוט)

1. לחץ על `index.html` למעלה
2. לחץ על אייקון העיפרון ✏️ (Edit)
3. מחק את כל התוכן
4. הדבק את הקוד החדש שקיבלת
5. למטה: **Commit changes** → **Commit directly to the main branch**
6. האתר יתעדכן אוטומטית תוך 1-2 דקות

### דרך 2: העלאת קובץ

1. לחץ על **Add file** → **Upload files**
2. גרור את `index.html` החדש (יחליף את הישן)
3. **Commit changes**

### דרך 3: דרך Git (למתקדמים)

```bash
git clone https://github.com/abenba/oncam-coach.git
cd oncam-coach
# החלף את index.html
git add .
git commit -m "Update app"
git push
```

## 🛠️ הגדרות GitHub Pages

אם זה לא עובד אחרי ההעלאה:

1. לך ל-**Settings** של הפרויקט
2. בתפריט צד שמאל: **Pages**
3. **Source**: בחר `main` branch → `/root` → **Save**
4. חכה דקה - תקבל הודעה: "Your site is live at..."

## 📱 שימוש באפליקציה

1. פתח ב-**Safari** (לא Chrome) באייפון
2. אשר הרשאות מצלמה + מיקרופון
3. בחר סוג אימון (פיץ', נאום, ראיון, אודישן)
4. לחץ "התחל הקלטה"
5. דבר מול המצלמה
6. "סיום הקלטה" → קבל ניתוח מפורט

---

## 🐛 פתרון בעיות

**המצלמה לא עובדת:**
- וודא שפתחת ב-Safari (לא Chrome/Firefox)
- בדוק הגדרות → Safari → אפשר גישה למצלמה
- רענן את העמוד

**הניתוח תקוע:**
- חכה 1.5 שניות - יש fallback אוטומטי
- אם מופיעה שגיאה - רענן ונסה שוב

**הקלטה לא נשמרת:**
- לחץ "שמור למכשיר" → יפתח בטאב חדש
- לחיצה ארוכה על הוידאו → "שמור וידאו"
